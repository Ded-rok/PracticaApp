# Информационная система «Распределение тем курсовых работ»

**Разработчик:** Гареев Вадим Ильдарович
**Практика:** Производственная практика  
**Дата:** март 2026

## Описание проекта

Приложение предназначено для автоматизации учёта и распределения тем курсовых и выпускных квалификационных работ в колледже.

## Функциональность

- Просмотр списка тем (таблица Topics)
- Добавление новой темы
- Редактирование существующей темы
- Удаление темы
- Поиск по названию

## Технологии

- C# .NET Framework 4.7.2
- WPF
- SQL Server (PracticaDB)
- ADO.NET

## Инструкция по запуску

1. Установи SQL Server Express.
2. Выполни скрипт `PracticaDB.sql` (приложен).
3. Открой `PracticaWpfApp.sln` в Visual Studio.
4. Нажми F5.
